# Changelog

本项目的重要变更记录在此文件中维护。

## 1.0.0 - 2026-07-20

- 完成商品、SKU、库存、购物车、订单、物流、售后和会员账户主流程。
- 完成优惠券实际优惠额核算，以及支付、退款、对账和 Outbox 状态恢复。
- 接入支付宝沙箱下单、异步通知、主动查单、超时关单和渠道退款流程。
- 完成知识版本发布、execution fencing、向量删除队列和 Milvus 重连恢复。
- 完成 AI Agent、工具调用、Pending Action、会话记忆、人工确认和 RAG 检索。
- 完成管理员 RBAC、操作审计、账户安全、HMAC 和生产危险配置封锁。
- 建立 Flyway 升级链、备份恢复、质量门禁、可观测性和发布验收工具。
- 增加商城、管理后台、Java、AI、数据设施和监控组件的完整 Docker 部署。

当前版本包含可运行的完整工程环境，但生产上线仍需配置真实外部凭据，并完成所在部署环境的支付、模型、邮件、备份、告警和故障恢复验收。
