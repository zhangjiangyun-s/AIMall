# Agent 完成报告模板

每个 agent 完成任务后，必须按下面格式回报。

```text
1. 所属目录：

2. 已完成内容：

3. 主要新增或修改文件：

4. 启动命令：

5. 运行端口：

6. 已实现接口或页面：

7. 自测结果：

8. 已知未完成事项：

9. 需要项目组长决策的问题：

10. 备注：
```

## 项目组长验收规则

我会按以下标准检查：

```text
1. 是否只在自己负责的目录内工作。
2. 项目是否必须能启动并通过自测。
3. 要求的接口或页面是否存在。
4. 是否遵守统一架构边界。
5. 是否违反禁止事项。
6. README 是否足够清楚，其他人能否按说明启动。
7. 未完成事项是否如实说明。
```

## 验收结果类型

```text
PASS：通过。
PASS_WITH_MINOR_FIXES：基本通过，有少量问题需要修。
NEEDS_REWORK：需要返工。
BLOCKED_BY_DECISION：被方案决策卡住，需要项目组长或用户决定。
```
